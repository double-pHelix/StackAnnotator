Set Up
-----------------------
**Environment Variables**

Must set up the following variables that can be accessed at https://apps.twitter.com/ (use the developer account)

 - `CONSUMER_KEY`
 - `CONSUMER_SECRET` 
 - `ACCESS_TOKEN`
 - `ACCESS_TOKEN_SECRET`


Example Usage
-----------------------

`from twitteraux import tweeter`

`tweeter.send_tweet("This is a tweet")`
